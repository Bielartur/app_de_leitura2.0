1 - Anotação sobre a chave de acesso

A ideia na parte de renovar o token de acesso é a seguinte, o JS vai fazer uma requisição
assíncrona para a view de access token, e enquanto a requisição não tiver tido alguma
resposta, irá aparecer o botão mais apagado e com um símbolo de loading rodando e o nome
"Carregando", o nome de quando não estiver enviando vai ser "Enviar". Caso a resposta
seja de sucesso a página deverá recarregar e aparecer um popup verdinho no canto da tela
confirmando que "A Chave de Acesso foi atualizada com sucesso".

2 - Quando o usuário colocar para adicionar um livro, pode acontecer 2 coisas. Caso o livro já
exista um registro no banco de dados, ele vai apenas referenciar aquele livro, como livro em
andamento daquele usuário. Caso contrário, ele vai realmente adicionar aquele livro no banco de
dados e também referenciar como livro em andamento para aquele mesmo usuário.
